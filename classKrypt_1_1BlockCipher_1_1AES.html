Detailed Description

Advance Encryption Standard.

provides method of encryption and decryption.

Warning
the method of this class only operates on a 16 byte array of unsigned char* or Bytes* if you pass an array of unsigned char* in the method with less than 16 elements in it that will cause an access overflow.
